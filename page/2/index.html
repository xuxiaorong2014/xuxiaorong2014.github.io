GitHub 使用方法

1.客户端下载地址： msysgit.github.io

2.下载项目： git clone https://xxx.xxx.git

3.将改动的地方添加到版本管理器 git add .

4.提交到本地的版本控制库 git commit -m “changes log”

5.提交到 github账号里 git push -u origin master
