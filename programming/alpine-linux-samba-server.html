Alpine Linux 设置 samba windows局域网文件共享

安装软件包:

# apk add samba

创建需要共享的目录:

# mkdir /mnt/sdb1
# chmod 0777 /mnt/sdb1

编辑配置文件 /etc/samba/smb.conf ，修改内容为：

[global]
   workgroup = WORKGROUP
   netbios name = ALPINE
   server string = Samba Server
   max log size = 50
   map to guest = Bad User
   guest account = nobody
   ;domain master = yes
   ;local master = yes
   ;preferred master = yes
   ;os level = 512
   ;wins support = yes

[share]
   comment = share
   path = /mnt/sdb1
   public = yes
   writable = yes
   printable = no
   create mask = 0765
   directory mask = 755
   browsable = yes

配置系统服务

# rc-update add samba
# rc-service samba start

然后在windows系统打开网上邻居，或者文件管理器地址栏直接输入 \ALPINE 就能访问共享了
